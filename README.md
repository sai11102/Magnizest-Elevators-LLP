# 🚀 Magnizest Elevators LLP - AMC Management System

A modern Elevator AMC (Annual Maintenance Contract) Management System built using Next.js, TypeScript, Tailwind CSS, Supabase, and Vercel.

## 📋 Features

### 👥 Customer Management
- Add, Edit, Delete Customers
- Customer Contact Details
- Search Customers

### 🏢 Site Management
- Add Customer Sites
- Manage Site Information
- Track Multiple Sites

### 🛗 Lift Management
- Lift Capacity Management
- Lift Type Management
- Door Type Management
- Floor Tracking

### 📄 AMC Management
- AMC Number Tracking
- AMC Start & End Dates
- Handover Date Tracking
- AMC Expiry Monitoring

### 🔧 Service Schedule Management
- Create Service Schedules
- Bulk Schedule Generation
- Upcoming Services
- Completed Services
- Missed Services
- Service Status Tracking

### 🔔 Notifications
- AMC Expiry Alerts
- Service Due Notifications
- Critical Alerts
- Dismiss Notifications

### 📊 Dashboard
- Customer Statistics
- Site Statistics
- Lift Statistics
- AMC Statistics
- Service Overview

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

Go to project folder:

```bash
cd YOUR_REPOSITORY
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build project:

```bash
npm run build
```

---

## 🔑 Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=YOUR_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```
